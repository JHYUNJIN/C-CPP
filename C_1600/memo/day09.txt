포인터 변수
	주소값을 저장하는 변수

배열 
	배열도 이름 자체가 주소값을 가지고 있기 때문에 포인터이다.
	그러나 배열의 경우 다른 주소값으로 변겨이 불가능하다.

상수 
	변수는 변수인데 속에 저장된 값을 변경할수 없는 상수이다.

	매크로 상수 #define Min 5 최상단에 선언
	상수는 대문자로 만들자
	const 상수화하는 함수
	ex) const int MAX = 100;

포인터 상수 (const pointer)
	포인터는 포인터인데 상수인 친구
	포인터인데 속에 있는 주소를 변경할 수 없는 포인터
	자료형* const 포인터명 = 주소값;
	

상수 포인터 (pointer to const)
	포인터는 포인터인데 상수를 가리키는 포인터
	포인터가 가리키는 대상을 상수처럼 취급하고,
	포인터 변수 속에 저장되어있는 주소값(변수)에 다른 값을 대입할 수 없다.
	const 자료형* 포인터명 = 주소값;

	//정리
	// 배열 방식으로 선언하면 요소(값)는 바꿀 수 있지만, 전체(주소)는 바꿀 수 없다.

	// 포인터 방식으로 선언하면 요소(값)는 바꿀 수 없지만, 주소는 바꿀 수 있다.
	//즉, 천체를 바꿀 수 있다.

포인터 배열 vs 배열 포인터

	포인터 배열 
		배열은 배열인데 포인터가 요소로 들어있는 포인터


	배열 포인터
		포인터는 포인터인데 배열의 주소값을 저장하는 포인터
