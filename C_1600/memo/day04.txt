연산자
	산술연산자

	관계(비교)연산자
		a > b	: a가 b보다 크니?
			맞으면 1, 아니면 0
	
		a < b

		a >= b

		a <= b
	
		a == b

		a != b	: a랑 b랑 다르니?
			맞으면 1, 아니면 0
	
	논리연산자
		a && b 	: 논리곱 and 연산	
			(a가 참이고, b도 참이면
			결과는 참, 아니면 거짓)
			a가 0이 아니고, b도 0이 아니면
			결과는 1, 아니면 0

		a || b 	: 논리합 or 연산	
			(a가 참 또는 b가 참이면
			결과는 참, 아니면 거짓)
			a 또는 b 둘 중 하나라도 0이 아니라면
			결과는 1, 아니면 0

		!a	: 논리부정 not 연산	
			(a가 참이라면 결과는 거짓
			a가 거짓이라면 결과는 참)
			a가 0이 아니라면 결과는 0
			a가 0이라면 결과는 1

		int num = 55;
		num > 50 && num < 100;

연산자의 우선순위
	최우선 연산자	()
	단항 연산자	!
	산술 연산자	+-*/%
	쉬프트 연산자
	관계 연산자	
          논리 연산자		&& || 
	삼항 연산자
	대입 연산자	=

	int num;
	num = num > 50 && num < 100;
						

	삼항연산자
		조건식 ? 참일 때 값 : 거짓일 때 값;

		조건식이 참이면 : 앞쪽 값이 연산의 결과로 나타난다.
		조건식이 거짓이면 : 뒤쪽 값이 연산의 결과로 나타난다.

조건식
	참 또는 거짓을 판별할 수 있는 식
	(0 또는 0이 아닌 숫자)

	대입연산자
